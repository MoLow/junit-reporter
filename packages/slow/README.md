# Slow tests Reporter
A Slow tests reporter for `node:test`.

## Installation

```bash
npm install --save-dev @reporters/slow
```
or
```bash
yarn add --dev @reporters/slow
```

## Usage

```bash
node --test --test-reporter=@reporters/slow
```
